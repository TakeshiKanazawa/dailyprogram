# 二分探索と対数時間

■線形探索法とは
例えば、数字(1-8)を当てるゲームがあったとして、
1から順に答えを行っていく方法のこと。
正解が「8」であった場合、8回の質問と回答が必要なため効率が悪い。

■二分探索法とは
選択肢の真ん中で条件を区切ること。
上記の例でいうと、最初に「数字は5以下ですか？」と
質問することにより、Yes/No どちらが答えとして帰ってきても
パターンが5通りにしぼられる。

２回目以降も同じように選択肢を半分に分ける質問をすることで
選択肢が8→4→2→1と減っていき、
どのようなケースでも3回の質問で答えを当てることができる。

これを二分探索法と呼ぶ

# 対数と指数おさらい
■指数とは
2^3 = 8
→２の右上に乗っている3のことを指数という
指数は、一つの数を何回かけるかを表す。
a^x = b

■対数とは
**を何乗すれば**になるかを表す数のことを対数と呼ぶ。
例えば2を何乗したら8になるかを表す数は以下のようになる
※2を底とする8の対数という
log2^8 = x

対数と指数は同じ数のことを差す。
視点と使い方が異なる。

x = loga^b
